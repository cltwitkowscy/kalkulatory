Design. Simulate. Build. Open-source Timber Engineering Toolkit

An open, modular platform for architects, engineers, manufacturers, and users to design and simulate CLT & timber buildings in any configuration.
Combine materials, calculate U-values and static loads, analyze layers, and generate detailed reports—fast.

See Demo ↓

Core Features

  • Interactive Layer Configurator
    Build any wall/roof/slab with CLT, timber, insulation, finishes.
  • Real-time Physics
    Calculate U-value, condensation, thermal and vapor profiles.
  • Preliminary Statics
    Check CLT panel span, thickness, loads, basic EC5 logic.
  • Ready-to-Use Presets & Hybrid Mode
    Use best-practice assemblies or mix systems (CLT+timber frame etc).
  • Reports & Exports
    Generate PDFs, data exports, and 2D/3D visualizations.
  • Open Data & API
    Built for extension by the timber engineering community.
